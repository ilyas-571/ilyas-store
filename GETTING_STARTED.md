## ❓ FAQ

**Q: Which guide should I read first?**
A: Start with [QUICK_START.md](./QUICK_START.md). It's the fastest way to get live.

**Q: Is it really free?**
A: Yes! Vercel + Railway free tiers are generous and suitable for small stores. No payment required.

**Q: What if I want a custom domain?**
A: All guides include custom domain setup. Domains are $0.88-15/year (cheap!).

**Q: Can I switch platforms later?**
A: Yes! Everything is built to be portable. You can move between platforms.

**Q: What if I get stuck?**
A: 
1. Check [DEPLOY_CHECKLIST.md](./DEPLOY_CHECKLIST.md) troubleshooting
2. Re-read the relevant section in your guide
3. Check platform dashboards for error logs
4. Google the error message

**Q: How long does deployment take?**
A: ~5 minutes per service (backend + frontend = 10 min total)
